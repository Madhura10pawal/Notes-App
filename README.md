# Notes-App

A simple React Notes Application that allows users to add, edit, and delete notes.

## Features
- Add, edit, and delete notes
- Uses React useState for state management
- Uses useRef for input focus
- Stores notes in localStorage
- Clean and simple UI

## Technologies Used
- React JS
- CSS
- localStorage

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
